# Data Engineering Portfolio

This repository contains my hands-on projects for the Data Engineering. Each folder corresponds to a module/exercise completed during the hands on experience.

## Modules
- docker/ - Environment setup with Docker
- sql/ - SQL exercises with Postgres / BigQuery
- pipeline/ - Python ETL scripts
- spark/ - PySpark batch transformations
- kafka/ - Streaming pipelines with Kafka
- dbt/ - Analytics engineering projects with dbt
- final-project/ - End-to-end data pipeline

## How to Run
Each folder has its own README explaining how to run the code and test the pipelines. Sample data and outputs are included where possible.
